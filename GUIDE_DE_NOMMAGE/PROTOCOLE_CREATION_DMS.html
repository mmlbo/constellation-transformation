3. Création du système

Étape 1 – Créer la structure des dossiers

$root = "C:\Users\a.adnane\constellation-dms"
$dirs = @("00_GOVERNANCE","01_TRANSFORMATION_ENGINE","02_DECISION_SYSTEM","03_AUDIT_READY","04_PMO","05_OPERATIONNEL_FINANCE","06_DATA_SYSTEM","07_KPI_REPORTING","REFERENTIEL_DOCS","GUIDE_DE_NOMMAGE","99_ARCHIVE")
New-Item -ItemType Directory -Path $root -Force | Out-Null
foreach ($d in $dirs) { New-Item -ItemType Directory -Path "$root/$d" -Force | Out-Null }

5. Règles de versioning

  • V1.0 : Première version
  • V1.1, V1.2 : Évolutions mineures
  • V2.0 : Changement majeur
  • V8.2.2 : Version alignée sur le document de transformation

6. Maintenance

  • Mettre à jour meta.json à chaque modification
  • Incrémenter la version dans le nom du fichier
  • Mettre à jour le index.html du dossier si nécessaire
  • Commit Git après chaque changement significatif
